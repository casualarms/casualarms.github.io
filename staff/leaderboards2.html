Season Report

Process for archiving monthly scores is as follows (games can be reported individually or simultaneously):

  1. Make sure all coin updates for the month are submitted and appear on the regular leaderboard website.
  2. Make sure an update for the subsequent month has not already been submitted. (If so, the tools cannot help you.)
  3. Enter the correct year and month in the form below and copy the JSON data in the field.
  4. Paste the JSON data in season-history.json as the first item in the list for the game in question.
  5. Go to the standard Leaderboard Tools and check the option to zero out the monthly coins. Submit the resulting JSON file as a normal leaderboard update.
